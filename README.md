# 白板画板工具 - Whiteboard Drawing Tool

一个功能完整的前端白板画板应用，支持多种绘图工具和实时预览，适用于教学、会议、创意草图等场景。

## 🎯 功能特性

### 核心绘图工具
- **画笔工具** - 自由绘制线条
- **橡皮擦** - 擦除已绘制内容
- **直线** - 绘制精确直线
- **箭头** - 绘制带方向箭头的直线
- **矩形** - 绘制矩形（支持空心/填充）
- **圆形** - 绘制圆形（支持空心/填充）
- **虚线** - 绘制虚线样式

### 高级功能
- **颜色选择器** - 自定义画笔颜色
- **笔刷大小调节** - 1-50px 范围调节
- **填充选项** - 形状填充开关
- **撤销操作** - 支持撤销上一步
- **清除画布** - 一键清空
- **保存图片** - 导出为PNG格式
- **触摸支持** - 完美支持移动设备

### 界面特性
- **全屏显示** - 无空白区域，完美适配
- **响应式设计** - 支持各种屏幕尺寸
- **实时预览** - 形状绘制时实时预览
- **简洁界面** - 直观易用的工具栏

## 🚀 快速开始

### 使用方式
1. 下载所有文件到本地文件夹
2. 用浏览器直接打开 `index.html`
3. 选择工具开始绘画
4. 完成后点击"保存"下载图片

## 🎨 使用指南

### 基础绘画
1. 选择画笔工具 (铅笔图标)
2. 调整颜色和笔刷大小
3. 在画布上拖拽鼠标/手指绘画

### 绘制形状
1. 选择对应的形状工具
2. 勾选"填充"选项可绘制实心形状
3. 在画布上拖拽鼠标绘制形状

### 高级操作
- **撤销** - 点击撤销按钮或按 Ctrl+Z
- **清除** - 点击清除按钮清空画布
- **保存** - 点击保存按钮下载PNG图片

## 🔧 自定义配置

### 画布尺寸
画布会自动适应窗口大小，也可通过修改 `script.js` 中的 `initCanvas()` 方法自定义尺寸。

### 默认设置
```javascript
// 默认颜色
currentColor: '#000000'

// 默认笔刷大小
currentSize: 5

// 默认工具
currentTool: 'brush'
```

## 🐛 常见问题

### Q: 保存的图片是空白的？
**A:** 确保画布上有内容后再保存，保存功能会自动添加白色背景。

### Q: 触摸设备无法绘画？
**A:** 确保设备支持触摸事件，大多数现代移动设备都支持。

### Q: 画布尺寸不正确？
**A:** 尝试刷新页面，画布会自动适应窗口大小。

### Q: 如何修改默认颜色？
**A:** 在 `script.js` 中修改 `currentColor` 的默认值。
